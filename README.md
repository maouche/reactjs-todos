# Todos App With ReactJs and Firebase

A simple Todos list application with ReactJS.

## Installation

1. `npm install` to install dependencies.

2. Create new project in [firebase](https://console.firebase.google.com/)

3. in the dashboard, add an application to start, click on web and save your application.

4. copy the variable firebaseConfig and replace it in src/components/firebase.js

5. In the project create a Firestore Database in test mode.

6. add a new collection called "todos".
